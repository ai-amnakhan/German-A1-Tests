# German A1 — Chapter Exams

A self-assessment quiz app covering the full German A1 course. Fourteen chapter exams, each with ~60 questions across four formats: multiple choice, true/false, fill-in-the-blank, and matching. No setup required — open any HTML file directly in a browser.

## Chapters

| # | Topic |
|---|-------|
| 1 | Alphabet, Pronunciation, Greetings & Introductions |
| 2 | Articles & Identifying der/die/das |
| 3 | Personal Pronouns, Plurals, Numbers, Months & Dates |
| 4 | Sentence Structure, haben/sein, Verbs (Präsens) & Negation |
| 5 | Asking Questions, Time Prepositions, Weather & Seasons |
| 6 | Adjectives, Adjective Endings & Colors |
| 7 | Modal Verbs (Präsens) |
| 8 | Past Tense (Präteritum): haben, sein & Regular Verbs |
| 9 | Perfekt & Partizip 2 |
| 10 | The Akkusativ Case & Akkusativ Pronouns |
| 11 | Possessive Articles & Pronouns (Nominativ & Akkusativ) |
| 12 | Introduction to the Dativ Case |
| 13 | Comparison (Komparativ/Superlativ) & Demonstrative Pronouns |
| 14 | The Imperativ (Commands & Requests) |

## Features

- **Instant feedback** — each answer is checked on the spot with an explanation
- **Progress bar** — tracks how many questions you've answered and how many are correct
- **Final score** — shown after the last question; pass mark is 80%
- **Dark / light theme** — toggle in the top-right corner
- **Fully offline** — answers are checked locally; nothing is saved or sent anywhere

## How to use

1. Open `index.html` in any modern browser
2. Pick a chapter from the grid
3. Answer each question and click **Check answer**
4. Aim for **80%+** before moving to the next chapter

## Tech

Plain HTML, CSS, and vanilla JavaScript. No dependencies, no build step, no server needed.
